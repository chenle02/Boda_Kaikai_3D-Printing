## Build it in Tinkercad (step-by-step)

### Bottom box

1. **Place a Box (solid)** → set size to **180 × 150 × 43 mm**.
2. **Place a Box (hole)** → set size to **170 × 140 × 35 mm**.
3. **Position the hole**:

   * Select both shapes → **Align** → center on X and Y.
   * Lift the **hole** up so its **top is flush with the top** of the solid (so the hole’s bottom sits 8 mm above the workplane).

     * Tip: Set the workplane on top of the solid, drop in the hole (35 mm tall), align, then **Group**.
4. **Group** to cut the cavity. You now have the bottom part with 8 mm floor and 5 mm side walls.

### Lid (outer slip-over cap)

1. **Place a Box (solid)** → set to **190.6 × 160.6 × 20 mm**.
2. **Place a Box (hole)** → set to **180.6 × 150.6 × 12 mm**.
3. **Position the hole**:

   * Put the workplane on the **bottom** of the lid solid.
   * Drop the hole so it sits on the bottom face (skirt depth 12 mm).
   * Select both → **Align** → center X and Y.
4. **Group**. This leaves an 8 mm top and a 12 mm-deep skirt with 5 mm wall thickness and 0.3 mm per-side clearance.

## Nice-fit & print tips

* **Test fit first:** print a tiny “corner test” (e.g., a 30×30 mm sample with the same walls/clearance) to confirm your printer’s fit.
* **Material notes:**

  * **PLA:** 0.2–0.3 mm clearance per side is often fine.
  * **PETG/ABS:** go **0.4–0.5 mm** per side because of more swell/elephant’s foot.
* **Elephant’s foot:** add a **0.3–0.5 mm bottom chamfer** (use “Shape Generators → Chamfer” or bevel) on the bottom part’s outer edge and the lid skirt’s inner edge to improve mating.
* **Fillets/comfort:** add small 1–2 mm fillets (rounding) on external edges for a smoother feel.
